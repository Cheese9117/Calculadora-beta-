# Calculadora

Calculadora web construida con **HTML, CSS y JavaScript** puro. Proyecto de práctica de frontend.

## Características

- Operaciones aritméticas básicas (suma, resta, multiplicación, división).
- Interfaz limpia y responsive.
- Sin dependencias.

## Uso

Abre `Calculadora/Calculadora.html` en el navegador.

## Estructura

```
Calculadora/
├── Calculadora.html
├── script.js
└── styless.css
```
